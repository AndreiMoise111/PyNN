====
PyNN
====

PyNN (pronounced 'pine') is a is a simulator-independent language for building neuronal network models.

The PyNN homepage is  http://neuralensemble.org/PyNN/

Development has now moved to `GitHub <https://github.com/NeuralEnsemble/PyNN/>`_ . All Trac tickets have been migrated to GitHub, although they do not in general have the same numbers. `Map from Trac tickets to GitHub issues <http://neuralensemble.org/trac/PyNN/wiki/Trac2GitHub>`_.
